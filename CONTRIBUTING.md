# Contributing to htl.ing

> Static site for the htl.ing domain.

## Development

```sh
npm install   # optional tooling
# Open index.html directly in a browser, or:
npx serve .
```

## Deployment

The site is served as a static page. Merges to `main` deploy automatically via GitHub Pages.

## Conventions

- Keep it simple: plain HTML + CSS, no heavy frameworks
- Commits follow [Conventional Commits](https://www.conventionalcommits.org/)
